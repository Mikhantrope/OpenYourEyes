Выручка (без НДС)
—
Сумма без налогов
«Сумма без налогов» из 1С = выручка без НДС
Правильная база для расчёта маржи и прибыли
Кол-во (шт)
—
целых единиц
«Кол-во реализации с возвратами» из 1С
Округлено до целого числа
Объём (кг)
—
кг = кол-во × вес SKU
Кол-во шт × вес единицы (из SKUСправочника)
Формат: 666 583.4 кг
Себестоимость
—
Стоимость (без НДС)
«Стоимость (без НДС)» из 1С
= прямые затраты на закупку товара
Прибыль
—
Выручка − Себестоимость
«Profit (сумма)» из 1С
= Сумма без налогов − Стоимость без НДС
⚠️ Валовая прибыль (ДО косвенных затрат)
Маржа %
—
Прибыль ÷ Выручка
Profit / Выручка без НДС × 100%
Возвраты
—
сумма возвратов
«Сумма возвратов» из 1С (отрицательное)
Возврат %
—
|Возврат| ÷ Выручка
|СуммаВозвратов| / Выручка × 100%
